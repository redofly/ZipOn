# ZipOn

ZipOn is a simple Python GUI tool built with Tkinter that lets you zip and unzip
files **and folders** easily.

## ✨ Features
- Zip a **file or an entire folder**
- Unzip `.zip` archives
- Simple and clean Tkinter interface
- Output folders are created automatically
- Python-only (no exe required)

## 📁 Output
- Zipped files are saved to the `zipped/` folder
- Extracted files are saved to the `extracted/` folder
